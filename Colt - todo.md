# 구현 기능 목록 - 콜트


## 1주차 - 홈 화면


### 홈 화면 : 투데이 탭
- 최초 메인화면에 추천 상품 더미데이터로 사전에 구성
- 추천 상품 리스트 조회(데이터 가공 및 출력을 최우선으로)    


### 홈 화면 : 상세 - 카테고리 선택 결과 화면
- 해당 카테고리 전체 결과 조회
    - 해당 카테고리 내의 상세 카테고리 분류 기능
    - 가격, 색상, 스타일 등 검색 조건 필터 기능


## 2주차 - 상세 정보 화면 구체화 / 구매하기 프로세스 완료


### 상세 정보 화면 구체화
- 상세 정보 화면 상단
- 리뷰 갯수, 상품 준비기간
- 셀럽마켓 이름과 태그
- 상세 정보 화면 중간
- 상품정보 탭
- 리뷰 탭 (더미데이터 출력)
- 문의 안내 탭은 텍스트 UI만 구성
